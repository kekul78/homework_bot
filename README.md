# homework_bot

Работал над созданием телеграм-бота, который раз в 10 минут опрашивал внешний API Практикума и, при обновлении статуса работы, отправлял сообщение в телеграм.

## Стек технологий

* Python 3.11.3,
* Python-telegram-bot  13.7,
* Requests 2.26.0
